# 模型折扣概览

通过官方优惠或第三方平台，以更低价格调用主流模型。

## 折扣来源

| 类型 | 说明 | 折扣力度 |
|------|------|---------|
| 官方 Batch API | 非实时批量请求 | 通常 5 折 |
| Prompt Cache | 重复内容自动缓存 | 最高节省 90% |
| 第三方中转平台 | 聚合多家模型 | 不定，需评估稳定性 |
| 新用户免费额度 | 官方赠送 | 固定金额 |

## 注意事项

- 第三方平台存在**稳定性风险**，生产环境慎用
- 注意折扣平台的汇率和手续费，实际成本可能更高
- 官方 Batch API 是最安全的降本方式
